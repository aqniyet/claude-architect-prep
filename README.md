# Claude Certified Architect - Foundations Exam Prep

Practice exam and targeted study materials for the **Claude Certified Architect - Foundations** certification.

## What's Inside

### [practice-exam.html](practice-exam.html)
Interactive 20-question practice exam covering all 5 domains:
- Domain 1: Agentic Architecture & Orchestration (27%)
- Domain 2: Tool Design & MCP Integration (18%)
- Domain 3: Claude Code Configuration & Workflows (20%)
- Domain 4: Prompt Engineering & Structured Output (20%)
- Domain 5: Context Management & Reliability (15%)

Answers are hidden until you submit. Includes per-domain scoring and a scaled score out of 1000 (720 to pass).

### [study-guide.html](study-guide.html)
4-chapter study guide that builds from basics to exam-level depth:
1. **How Claude Code Is Configured** - the 4 configuration surfaces, shared vs personal tiers, path-scoped rules, skills frontmatter, CI flags
2. **Getting Structured Data Out of Claude** - tool_use as structured output, tool_choice control, schema design, validation-retry, Batch API
3. **Designing Tools and Distributing Them** - tool descriptions, the tool count problem, MCP server config, structured errors
4. **Managing Context and Information** - session management, fact extraction, provenance tracking

## How to Use

Open the HTML files directly in your browser - no build step or server needed.

1. Take the practice exam first to identify your gaps
2. Study the chapters that match your weak domains
3. Retake the exam to measure improvement

## Exam Overview

| Detail | Value |
|--------|-------|
| Format | Multiple choice (1 correct, 3 distractors) |
| Passing score | 720 / 1000 |
| Scenarios | 4 of 6 randomized per exam |
| Domains | 5 (weighted as listed above) |
